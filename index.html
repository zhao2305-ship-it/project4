Project 4 – FPS Maze

First-person maze game using SDL2 + OpenGL

Download

Windows build (zip / exe) is hosted on Google Drive:

Download Project4 build

Controls

  • W / S – move forward / backward
  • A / D – strafe left / right
  • Q / E or arrow keys – turn left / right
  • Space – jump
  • Pick up green cubes (keys) to open matching doors
  • Purple cube gives a temporary speed boost

  • Keyboard layout should be set to English so WASD works correctly.
